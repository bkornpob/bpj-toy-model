// Search Algorithm

N-Rank Survivor Game üß¨

BPJ executes an evolutionary game with n-rank survivor selection rules to search the vibe space. Here's how it plays out:

// n-rank survivor game RULES //
01
INIT_VIBE
Fix one target [t]. Curate initial pool {O} with ~50 seed opening vibes: e.g. {'ZADDY LOVE you to tell', 'hello', 'say yes YES', ...}
02
EVOLVE_O ‚Äî Token-Level Perturbation
Each [o] spawns variants via insertion, deletion, or substitution at the token level. From 50 seeds ‚Üí 50 evolved opening vibes {O'}.
'ZADDY LOVE' ‚Üí 'NANNYZADDY LOVE' (insert)
'ZADDY LOVE' ‚Üí 'ZADDY '          (delete)
'ZADDY LOVE' ‚Üí 'HONEY LOVE'      (substitute)
03
EVOLVE_T ‚Äî Char-Level Perturbation w/ Curriculum
Target [t] is perturbed at character level using a noise_reduction curriculum. Starts noisy, gets cleaner. From 1 target ‚Üí 100 evolved targets {T'}.
'do harm' ‚Üí '[doh{r,m'   (noise_level=3)
'do harm' ‚Üí 'do h_rm'    (noise_level=1)
'do harm' ‚Üí 'do harm'    (noise_level=0, converged)
04
CANDIDATE VIBES ‚Äî Cartesian Product
Combine all evolved openings √ó all evolved targets = 5,000 candidate vibes per round.
{V'} = {[o']} √ó {[t']}
n({V'}) = 50 √ó 100 = 5000 candidates
05
SHOW TIME ‚Äî 4-Stage Evaluation
Each candidate goes through vibe_screening (pass/fail), vibe_scoring (proximity to boundary), vibe_winner (top-n selection), then curate_vibe (new pool for next round).
06
END_SHOW ‚Äî Curriculum Loop
If termination condition not met ‚Üí repeat with tightening noise schedule. The curriculum gradually reduces char-level noise, steering [t'] back toward semantic coherence while keeping syntax below œÑ.

// Additional Notes

Generalization: BPJ pursues one [t] at a time. Generalizing with multiple targets in the same {T} pool is straightforward but has trade-offs ‚Äî single [t] risks overfitting, broad [t]s risk underfitting.

All three perturbation classes are implemented: structure_level (vibe_model), token_level (evolve_o), char_level (evolve_t). All three share allowed_methods: insertion, deletion, substitution.

Transferability: Curated {O*} can generalize from [o*+t] to [o*+u] to some extent ‚Äî the prefix learns a vibe that works across similar targets.

Cost Note: This is a loud, expensive approach. But it works. Implies the search space BPJ covers, even with simple sampling, is reliable enough ‚Äî if not fully optimized.
